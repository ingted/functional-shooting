=========================
shootingserver
=========================

pbsk-functional-shootingのserver実装

ビルド・実行環境
-------------------

Erlang/OTP(>= R15B、R15B02推奨)がインストールされたLinux,Unix環境。

(現状では、Windowsだとjiffyのビルドに失敗する)

環境は随時最新リリースに対応していく予定。

使用ライブラリ
--------------------

- cowboy
- jiffy
- lager
